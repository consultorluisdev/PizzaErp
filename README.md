# 🍕 PizzaERP

Sistema completo de gestão para pizzaria, com dashboard, autenticação e controle de pedidos.

---

## 🚀 Tecnologias

* ⚙️ Backend: .NET (Web API)
* 🎨 Frontend: React + Vite + TailwindCSS
* 🐘 Banco de Dados: PostgreSQL
* 🐳 Docker (em breve)

---

## 🔐 Funcionalidades

* ✅ Login e Registro de usuários
* ✅ Dashboard com métricas
* ✅ Sistema de autenticação JWT
* 🚧 Rastreamento de entregas (em desenvolvimento)
* 🚧 PDV (em desenvolvimento)

## 🛠️ Como rodar o projeto

### 🔹 Backend

```bash
cd backend
dotnet run
```

---

### 🔹 Frontend

```bash
cd frontend/pizza-web
npm install
npm run dev
```
